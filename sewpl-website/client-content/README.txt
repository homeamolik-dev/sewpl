CLIENT WEBSITE CONTENT FILES

These files control the text content for the website.
Please edit only the words inside quotation marks.
Do not remove commas, brackets, or quotation marks.

Which file changes what:

- site-global.json
  Header menu, footer text, shared labels, website title/description

- home-content.json
  Homepage text

- about-content.json
  About page text

- services-page-content.json
  Services page headings and labels

- contact-page-content.json
  Contact page headings

- contact-form-content.json
  Contact form labels, placeholders, button text, success message

- company.json
  Company details, contact details, stats, mission, vision, social links

- services.json
  Service descriptions, facility description, highlights, capabilities

Important:

- Please do not rename the files
- Please do not change the field names on the left side
- Please do not edit product details yet
- If you are only changing wording, edit the text inside quotes only

Good example:
"title": "New Heading Here"

Please send the edited files back in the same folder structure.
